Knitting and crochet designs

Knitting designs

Each knitting design instructions are provided in downloadable PDF file. It consists of step by step tutorials with photos. It also includes graphic charts and colour charts.
All descriptions are detailed and suitable both beginner and advanced knitters. If you prefer row by row descriptions, it is included too.

Crochet designs

Each crochet design instructions are provided in downloadable PDF file. It consists of step by step tutorials with photos. It also includes graphic charts and colour charts.

How it works

You will get instant PDF file download. Once you added your items to basket and paid using Paypal or your credit/debit card, you will get email with a link to downloadable PDF file. Please click the button ‘Download’, save the file and can start to knit or crochet straight-away. It might be useful to save to your cloud account, if you have one, that you can access your file from any of your connected devices.
